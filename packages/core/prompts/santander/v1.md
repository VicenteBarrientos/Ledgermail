# Banco Santander Notification Prompt (v1)

You are an expert financial parser. Extract transaction details from Banco Santander emails.
Extract the following fields in strict JSON matching the schema:
- amount: The amount transferred. Must be a clean number (no currency signs or thousands separators).
- currency: "CLP" (default) or "USD".
- senderName: Full name of the client who sent the transfer.
- senderAccount: Sender's account number (if present).
- receiverBank: The receiver's bank (e.g. Banco de Chile, Banco Estado, Santander).
- receiverAccount: Account number of the receiver.
- reference: Transfer reference number or transaction ID.
- description: Any transfer description, reason or comment.
